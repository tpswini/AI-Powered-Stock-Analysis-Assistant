# AI-Powered Stock Analysis Assistant

## Overview

AI-Powered Stock Analysis Assistant is a Streamlit-based application that combines stock market data analysis with Large Language Models (LLMs). The system retrieves stock data, generates visualizations, and provides AI-assisted insights through a conversational interface.

## Features

* Stock data retrieval using Yahoo Finance
* Historical stock performance analysis
* Interactive charts and visualizations
* Natural language stock queries
* AI-generated market insights
* PDF report generation
* User-friendly Streamlit interface

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Seaborn
* YFinance
* Ollama
* Mistral LLM
* ReportLab

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-stock-analysis-assistant.git

cd ai-stock-analysis-assistant

pip install -r requirements.txt

streamlit run app.py
```
